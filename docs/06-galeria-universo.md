# 06 · Sección 4: Galería del Universo (Arte Oficial)

## Layout

- Contenedor **centrado** para la galería interactiva.

## Mecánica

- Carrusel ligero: **Splide.js**.
- Las imágenes secundarias tienen **opacidad reducida**.
- La imagen en foco encaja mediante **transición de fade suave** — evitar desplazamientos mecánicos bruscos.

## Estética

- **Marcos negros muy finos** alrededor de las ilustraciones, emulando un portafolio de arte físico.

## Rendimiento

- Todas las imágenes de la galería llevan `loading="lazy"` (ver [09-animaciones-y-tecnica.md](09-animaciones-y-tecnica.md)).
